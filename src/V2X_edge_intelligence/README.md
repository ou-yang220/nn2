# 车路协同与边缘智能

**核心内容**：结合车路协同（V2X），用边缘计算/联邦学习提升单车感知与决策能力，降低对车载算力的依赖。

**技术路径**：

- **road_side_perception_enhancement**：路侧部署轻量级模型（YOLOv7-Tiny），向车辆推送盲区信息，提升全局态势感知。

- **edge_vehicle_cooperative_inference**：复杂计算（如3D检测）在路侧完成，车辆仅做实时控制。

- **federated_optimization**：多车联合训练，在不共享原始数据的前提下提升模型精度。

**大学生项目**：“基于V2X的校园无人车协同避障系统”

- 采用ROS+5G模块实现车路通信，验证多车协同的通行效率。
